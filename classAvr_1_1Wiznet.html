Detailed Description

Avr::Ethernet driver for WIZnet Ethernet modules (W5100, W5200, W5300, W5500).

The usage is described in the documentation of the Avr::Ethernet module.

Note
The implementation is based on the WIZnet ioLibrary (https://github.com/Wiznet/ioLibrary_Driver).
